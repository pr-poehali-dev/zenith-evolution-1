# Stride Collective - Беговое Сообщество

![Preview](https://cdn.poehali.dev/templates/stride-collective/preview-ru.jpg)

Современный лендинг для бегового клуба или спортивного сообщества. Включает hero-секцию с эффектным визуалом, секцию миссии с градиентной анимацией текста при скролле, timeline с историями участников, отзывы с stagger-анимацией и финальную секцию с parallax-эффектом. Построен на React + Tailwind + Framer Motion.
